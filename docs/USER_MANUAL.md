# User Manual

1. Select Patient / New User or Doctor in the sidebar.
2. Register first, then log in with the same email and password.
3. Patients can open Patient Portal, upload a CSV/TSV/TXT cancer report, and get AI-assisted output.
4. Patients can also enter report values manually and consult doctors from the same portal.
5. Doctors can open Doctor Workspace to review analytics, patient records, prediction history, and directories.
6. Select `best`, Logistic Regression, SVM, or Random Forest for predictions.
7. Review prediction result, confidence score, and feature contribution chart.
8. Search Doctors Directory by city or specialization.
9. Book or cancel appointments.
10. Use Healthcare Support for awareness, symptoms, prevention, nutrition, mental health, and emergency guidance.

Clinical note: AI predictions are decision-support outputs and must be reviewed by qualified clinicians.
